Calendar Feed Integration Setup

This events page is designed to automatically pull and display events from multiple calendar sources:

  • Trail Life Connect: Pull regional events directly from your TL Connect calendar
  • Troop Calendars: Aggregate events from PA-1997, NY-2911, and other troop calendars
  • Auto-Sync: Events update automatically as calendars change
  • Color Coding: Events are color-coded by source for easy identification

To enable calendar integration, you'll need:

  1. iCal feed URLs from Trail Life Connect and each troop calendar
  2. Add these URLs to the js/calendar.js configuration file
  3. The calendar will automatically fetch and display events
